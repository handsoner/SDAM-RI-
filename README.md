# SDAM-RI : Semi-supervised Distribution-Aware disease diagnosis model based on Multimodal fusion of Raman spectroscopy and Infrared spectroscopy

The code of "Single-source Domain Adaptive disease diagnosis model based on Multimodal fusion of Raman spectroscopy and Infrared spectroscopy" The code will be made publicly available within one week after the paper is officially accepted.

## Abstract

The integration of artificial intelligence with vibrational spectroscopy, such as Raman and infrared, for disease diagnosis is rapidly emerging. However, these data-driven approaches heavily rely on large amounts of expertly annotated spectral data to achieve high diagnostic accuracy. The annotation process is not only exceptionally time-consuming but also consumes substantial medical resources. To reduce the heavy reliance on annotated medical data resources, this study proposes a Semi-supervised Distribution-Aware disease diagnosis model based on Multimodal fusion of Raman spectroscopy and Infrared spectroscopy (SDAM-RI). The model aims to enhance generalization and accuracy in unlabeled data scenarios. The proposed framework innovatively integrates two complementary vibrational spectroscopies—Raman and infrared—through a co-attention mechanism to deeply fuse cross-modal features. The domain adaptation framework disentangles domain-specific and domain-invariant information, employing adversarial training and contrastive learning to minimize distributional discrepancies between the source and target domains. Multimodal fusion provides a more comprehensive biochemical representation, while the adaptation mechanism aligns cross-domain features in the fused subspace, jointly improving robustness and generalizability. Experiments on cancer and autoimmune disease spectroscopy datasets demonstrate that SDAM-RI significantly outperforms multiple classical baselines, achieving superior diagnostic accuracy in target domains. Visualization analysis further confirms the model's ability to learn highly discriminative features from spectral data. By focusing on biologically relevant spectral fingerprints, SDAM-RI provides a powerful and general solution to the challenge of limited annotations in multimodal vibrational spectroscopy data. The code is publicly available at https://github.com/handsoner/SDAM-RI.
